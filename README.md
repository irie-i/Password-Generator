# Password-Generator

## Project Description
 
 This project is designed to generate a password between 8 - 128 characters including: uppercase letters, lowercase letters, numbers, and special characters depending on the users needs/choice. By using creating arrays, using 'if/else' and 'for' statements that creat loops which create the desired password. 

## Instruction

1. Click "Generate Password"

2. Enter number of desired characters (between 8 -128 characters)

3. Confirm if upppercase characters are recquired

4. Confirm if lowercase characters are recquired

5. Confirm if numbers are recquired

6. Confirm if special characters are recquired

7. YOU'RE DONE! Password is generated!
